# Customer-churn-project
A project to predict whether a customer will leave(churn = yes), or stay(churn = no). This project may be used to help an organization reduce customer loss, increase revenue retention and identify risk customers early.
## Tools Used
- Python
- Pandas
- Scikit-learn
- XGBoost
- Matplotlib

## Models Used
- Logistic Regression
- Random Forest
- XGBoost
